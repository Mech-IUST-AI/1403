Road Lane Detection Project
Goal
Detect lines, roads, and cars in video footage using deep neural networks.

How to Use
Download the model from the repository below and place it in the "models" folder.

Prepare your video.

Run video_road_detection.py.

Save the output.

Run Lane_detection.py. The input for this script is the output from the previous step.
